InBody 대응전략

⚙️

기술 차별화

  • 경쟁사 경험적 추정 방식의 한계 강조
  • 세그멘탈 데이터 (부위별 위상각/수분) 우위
  • 의료 기반이면서도 직관적인 결과 해석
  • 종이 리포트 출력 옵션 (vs SECA SW 전용)
💰

가격 · 번들 경쟁력

  • 낮은 SW 구독 비용 (vs Evolt $99)
  • Zalo/로컬 SW 연동 (베트남)
  • 매스마켓 장비사 협업 (MBH/DHZ)
  • 파이낸싱 옵션 검토 (vs Evolt 60개월 할부)
🤝

파트너십 · 서비스

  • Life Fitness: InBody에 먼저 제안
  • EGYM: 본사 방문 연동 완료
  • 적극적 애프터세일즈 교육·이벤트 지원
  • Precor: 프리미엄 채널 제휴 (베트남/대만)
⚖️

IP · 법적 리뷰

  • 경쟁사 제품의 특허 침해 여부 검토 필요
  • 결과지 항목·알고리즘 지적재산권 분석
  • 경쟁사 마케팅 클레임의 법적 대응 가능성 검토
  • 현재 미착수 — 선제적 IP 전략 수립 시급
Key Insight

Evolt의 파이낸싱 공세(0% 60개월)와 Anovator의 저가+분납 전략에 대응하려면 InBody도 유연한 결제 옵션 검토가 필요

  • 피트니스 장비 업체와의 연동/통합이 파트너십의 핵심 형태
